Cómo comprar

Para coordinar la compra y asegurar el stock para compras de varias unidades debe seguir las instrucciones:

  1. Debe completar el siguiente formulario, asegurerse de ingresar los datos de forma correcta y completa
  2. Nos contactaremos con usted desde el correo ventas@fotofenix.com, Foto Fénix no se contactará con ningúna otra dirección correo, incluso de la misma compañia, para tratar ventas
  3. Una vez confirmado el stock del producto meidiante un correo desde ventas@fotofenix.com, se le entregará un codigo de compra que debe adjuntar como comentario en la trasferencia o a través de nuestra pasaréla de pago cuandos se le solicite
  4. Confrimada la compra se hará envío del pedido, recibirá un correo desde ventas@fotofenix.com con el código y las instrucciones para realizar el seguimiento del pedido
